# AI Programming Foundations - MSc Capstone Project

## Project Overview

This repository contains the Udacity MSc capstone foundation project focusing on AI programming fundamentals using Python and Jupyter notebooks.

## Project Structure

```
├── notebooks/          # Jupyter notebooks for analysis and development
├── src/                # Reusable Python modules
├── data/               # Data directory (raw and processed)
├── models/             # Trained model artifacts
├── results/            # Output results and visualizations
├── tests/              # Unit tests
├── docs/               # Project documentation
└── config/             # Configuration files
```

## Setup Instructions

1. **Clone the repository:**
   ```bash
   git clone https://github.com/8yfw8v5r7v-png/ai-programming-foundations-project.git
   cd ai-programming-foundations-project
   ```

2. **Create a virtual environment:**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run Jupyter notebooks:**
   ```bash
   jupyter notebook
   ```

## Requirements

See `requirements.txt` for all dependencies.

## Getting Started

1. Start with notebooks in the `notebooks/` directory
2. Add reusable code to modules in `src/`
3. Store results and figures in `results/`
